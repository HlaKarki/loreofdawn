==Meta Mid laners==
It is recommended that you practice each Mid lane hero at least once, so that you know enemy skill types, cooldowns, and learn matchups and recommended skill builds for different heroes. A Mid laner will determine the pick off and ganking routes (because mid laners usually rotate with the roamer).

These heroes are currently meta and will dominate (most of the time as shown in tournaments) if you mastered them (In no particular order):{{Hero-box-xs|name=Pharsa}}{{Hero-box-xs|name=Valentina}}{{Hero-box-xs|name=Luo Yi}}{{Hero-box-xs|name=Zhuxin}}{{Hero-box-xs|name=Kimmy}}{{Hero-box-xs|name=Yve}}
